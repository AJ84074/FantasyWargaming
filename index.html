Character Creation

Attributes

Attributes are rolled in the order listed below, using 4d6 and keeping the highest three dice for a range of 3 to 18.

Roll 4d6 and keep the highest three dice for the following attributes:

  • Physique
  • Agility
  • Endurance
  • Intelligence
  • Faith
  • Charisma
  • Appearance
  • Bravery
  • Greed
  • Selfishness
  • Lust

The maximum value for any attribute is 20, not counting points from any bogey.

Social Class
